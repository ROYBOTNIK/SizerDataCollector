# 01 Intake

## Inputs

- Layer 4: `../../inbox/*.md`
- Layer 3: `../../_config/backlog-rubric.md`

## Process

For each note, create one backlog item in `../../backlog/items/`. Preserve useful user wording, add priority, protected-action status, and acceptance checks.

If a note is not actionable, leave it in `inbox/` with a short reason appended.

## Outputs

- `../../backlog/items/YYYYMMDD-short-title.md`
