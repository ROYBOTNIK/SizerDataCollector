# 03 Plan

## Inputs

- Layer 4: one `../../backlog/items/*.md`
- Layer 3: `../../_config/backlog-rubric.md`
- Layer 3: relevant repo docs from `../../../MD-DOCS/`

## Process

Choose one item. Identify the root cause, callers, files, tests, and protected actions. Consider `A = do nothing`, `B = minimal change`, and `AB = broader change`.

## Outputs

- A short plan in `output/<item-id>.md`, or update the backlog item directly when the plan is trivial.
