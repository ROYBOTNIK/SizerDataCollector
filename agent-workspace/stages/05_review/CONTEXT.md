# 05 Review

## Inputs

- Layer 4: repo diff
- Layer 4: selected backlog item
- Layer 3: `../../_config/backlog-rubric.md`

## Process

Run the smallest relevant checks. Review for correctness, data loss, security, lifecycle, docs drift, and missing tests. Compare the final diff against doing nothing.

## Outputs

- `output/<item-id>-review.md` with checks, findings, decision, and residual risk.
